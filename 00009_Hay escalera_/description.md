Queremos saber si dadas dos cartas de un mazo español (con 8's y 9's), hay un escalera.

 
> ```javascript
> hayEscalera(1,5)
> false
>
> hayEscalera(9,10)
> true
>
> hayEscalera(5,4)
> true
>
> hayEscalera(1,1)
> false
> ```

Programá la función `hayEscalera` que toma como argumentos dos enteros y retorne un booleano.
