Queremos saber si dadas tres cartas de un mazo español (con 8's y 9's), hay una escalera.

 
> ```javascript
> hayEscalera(1,5,10)
> false
>
> hayEscalera(9,10,8)
> true
>
> hayEscalera(5,4,3)
> true
>
> hayEscalera(1,1,10)
> false
>
> hayEscalera(12,1,11)
> false
> ```

Programá la función `hayEscalera` que toma como argumentos tres enteros y retorne un booleano.
